# 🏃 三人接力计时器

一个命令行下的三人接力赛计时工具，支持选手依次接力、犯规/超时检测、结果统计。

## 功能特性

- **顺序接力**：三位选手依次完成各自阶段，前一位未交棒时，下一位不能开始
- **实时计时**：每秒更新一次当前选手的用时显示
- **犯规检测**：输入 `f` 可标记当前选手犯规，接力立即停止
- **超时检测**：每位选手有独立时间限制（默认 30 秒），超时自动停止接力
- **异常处理**：优雅处理 Ctrl+C、Ctrl+D 等用户中断操作，不会异常退出
- **结果统计**：输出每段耗时、失败原因、未出场选手及总用时

## 使用方法

### 基本运行

```bash
python3 relay_timer.py
```

### 自定义选手名称

```bash
python3 relay_timer.py 张三 李四 王五
```

### 自定义时限

修改 `relay_timer.py` 中的 `time_limit_per_runner` 参数（默认 30 秒）。

## 操作说明

| 操作 | 说明 |
|------|------|
| `回车` | 开始当前选手的计时 |
| `s` + `回车` | 当前选手成功完成，交棒给下一位 |
| `f` + `回车` | 当前选手犯规，接力立即停止 |
| `Ctrl + C` | 手动中断，记为犯规 |
| `Ctrl + D` | 结束输入，记为用户取消 |
